Optimizing Deep Learning Architectures for Accurate image Segmentation in Medical Imaging

In this project, I investigated how different deep learning techniques can improve the accuracy of tumor segmentation in 2D MRI brain images. I compared two popular neural network architectures, ResNet-18 and VGG-16, to see which performed better for this task. I also tested different methods to improve the model, including different modifications of the network structure and data augmentation to increase the diversity of the training data. In addition, I compared two types of loss functions that guide the model's optimization process. The results showed that ResNet-18 outperformed VGG-16. Using certain advanced techniques, such as dilated convolutions, improved segmentation quality, and data augmentation helped the model generalize better to new images. Overall, these adjustments resulted in more accurate and reliable tumor detection in MRI scans.

Python, Deep Learning, PyTorch, Image Segmentation

Comparing Deep Learning Architectures for Accurate Object Classification

In this project, I evaluated four deep learning models, including VGG16, ResNet50, Xception, and MobileNet, which have different sizes and structures, to identify the most effective model for classifying objects in 2D MRI brain images. To improve the performance of the models, I applied techniques such as data augmentation and normalization. Each model was then fine-tuned using advanced optimization techniques and loss functions. I found that selecting the most appropriate model depends on the specific requirements of the task, with each model offering a different trade-off between accuracy and computational efficiency.

NLP Sentiment Analysis

In this project, I explore sentiment analysis using a variety of natural language processing (NLP) techniques. I evaluate four text preprocessing methods, including basic preprocessing, stemming, part-of-speech (POS) tagging, and bag-of-words vectorization. My analysis shows that relying on word frequency alone is not sufficient for effective vocabulary building. I also highlight the importance of exploring TF-IDF and addressing data imbalances to improve sentiment analysis.

Customer Churn Prediction

In this project, I developed and validated a customer churn prediction model using a machine learning approach for an advertising company, achieving 85% accuracy on their imbalanced dataset. I compared Logistic Regression, SVC, XGBClassifier, and a Forward Neural Network (FNN). I also performed hyperparameter tuning to improve performance. Logistic Regression and XGBClassifier showed the best results, the FNN also performed well, while SVC was less effective at handling class imbalances.

Python, Machine Learning, Deep Learning, Imbalance Data
